# Digital-Forensic-Report-

This project is a static website built using HTML and CSS.
It focuses on creating a clean layout, responsive design, and visually appealing user interface.
This project is ideal for beginners, college mini projects, and portfolio use.

## Features

Static website built with pure HTML & CSS

Responsive layout

Clean and simple UI design

Easy to understand folder structure

Fast loading and lightweight

Beginner-friendly code

## Technologies

HTML5

CSS3

## How to Run the Project

Download the ZIP file

Extract the ZIP file

Open the project folder

Double-click on index.html

The website will open in your browser
